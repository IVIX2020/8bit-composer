# 8-Bit Web Composer

ブラウザで動作する、レトロな8ビットサウンドの作曲ツールです。Web Audio APIを活用し、手軽にメロディやドラムパターンを作成・保存できます。

## 主な機能
- **マクロ/ミクロ構成**: 全体構成（タイムライン）とフレーズ編集（ステップシーケンサー）の分離。
- **相対度数ベースの作曲**: キーやスケールを変更しても、メロディの「性格（度数）」を維持したまま響きを変えられる音楽的アーキテクチャ。
- **リズムの自由度**: ステップ数（4〜32）と単位（4分〜16分音符）をフレーズごとに設定可能。
- **キャッシュ機能**: 作成した曲をブラウザに自動ストック保存（お気に入り機能付き）。
- **モバイル最適化**: タッチ操作に特化したUIと、画面を広く使える折りたたみ式コントロールパネル。

## 使い方
1. `index.html` をブラウザで開きます。
2. 「ミクロ (ピース編集)」タブでフレーズを編集します。
3. 「マクロ (全体構成)」タブでトラックにピースを配置します。
4. 「PLAY」ボタンで演奏します。
5. メニューから「保存」ボタンでブラウザに曲をストックできます。

## 開発環境
- 純粋なHTML/CSS/JS (Vanilla) で構成されています。特別なビルドツールは不要です。
